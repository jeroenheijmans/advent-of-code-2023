# Advent of Code 2023

Solutions to the [Advent of Code 2023](https://adventofcode.com/2023) edition by Jeroen Heijmans.

## Running

```psh
./run.ps1
```

or

```sh
deno run --watch --allow-read ./src/main.ts
```
